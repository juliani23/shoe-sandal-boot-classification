# Shoe vs Sandal vs Boot Classification

## About Project

This project was developed as part of the Dicoding [Deep Learning/Image Classification](https://www.dicoding.com/academies/185/corridor) course submission.

## Project Overview

This project aims to classify images into three categories: shoe, sandal, and boot using Convolutional Neural Network (CNN).

The trained model is exported into multiple formats including TensorFlow SavedModel, TensorFlow Lite, and TensorFlow.js.

## Project Structure

data/ → dataset files

notebooks/ → notebooks for training and experimentation

models/ → trained model files

outputs/ → visualizations and evaluation results

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
